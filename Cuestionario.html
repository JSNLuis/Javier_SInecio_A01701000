Cuestionario 1: BD vs DBMS

¿Qué es entonces una base de datos - DB?

Una base de datos es una colección de archivos interrelacionados, debemos de ser precavidos que la base contenga la menor repetición posible de datos hechos, abarcando las necesidades de la empresa.

¿En qué casos es conveniente usar bases de datos?

La base de datos es necesaria cuando en una sociedad, empresa, comunidad o conjuntos de personas con un mismo fin, necesiten la interrelación de archivos y que usuarios o aplicaciones necesiten el acceso a esta información, es necesario crear una base datos.

¿Qué es un sistema de gestión de base de datos - DBMS?

Es una colección de datos interrelacionados, así como, un conjunto de programas para entrar a estos mismos datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Proporcionar entorno de extracción de información, para que se conveniente y eficiente para ser utilizado al extraer y almacenar información. • Gestión de información, están diseñados para una gran manipulación de información. • Definición de estructuras, para el almacenamiento de información. • Provisión de mecanismos, para la gestión de información. • Mantener seguridad de la información guardada en la base de datos.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Pues han sido proyectos laborales, como trabajar en TI utilizamos mucho un almacén y no teníamos una base datos del stock que había, la razón es que la mayoría requería del stock, pero la administración y control de este era ineficiente, si querían saber quién había tomado algo del stock tenían que buscar entre papeles y esto era muy ineficiente, con una base de datos las entradas y salidas estarían controladas.
